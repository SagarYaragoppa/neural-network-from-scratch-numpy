# Neural Network from Scratch for Digit Classification

A simple neural network built from scratch using NumPy to classify handwritten digits.  
This project focuses on understanding the core concepts of neural networks without using any machine learning libraries.

---

## Features
- Neural network implemented from scratch
- Forward propagation and activation functions
- Loss and accuracy calculation
- Training visualization using Matplotlib
- No ML frameworks used

---

## Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib
